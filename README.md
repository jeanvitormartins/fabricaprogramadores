# 🚀 Minha Jornada na Programação: Da Lógica ao Python

Bem-vindo ao meu repositório de estudos! Aqui estão reunidos os meus primeiros passos no mundo do desenvolvimento de software. Este espaço funciona como um portfólio da minha evolução técnica, documentando desde os conceitos fundamentais de lógica até a criação de scripts automatizados.

---

## 🧠 O Que Eu Já Aprendi Até Agora

Minha base foi construída focando em entender como os computadores pensam, para só depois começar a ditar as regras em código. Aqui está o meu mapa de habilidades:

### 1. 📝 Algoritmos & Portugol
* **O início de tudo:** Aprendi a quebrar problemas complexos em passos lógicos e sequenciais.
* **Portugol:** Utilizei a pseudo-linguagem para entender estruturas de decisão e repetição sem me preocupar com as regras rígidas de uma linguagem de mercado. Foi onde a lógica ganhou forma.

### 2. 🔀 Fluxogramas
* **Visualizando o código:** Aprendi a desenhar o caminho que a informação percorre dentro de um sistema utilizando blocos visuais, facilitando o mapeamento de condições antes mesmo de começar a programar.

### 3. ➕ Operadores
* Dominio sobre os motores que fazem a lógica acontecer:
  * **Aritméticos:** Cálculos matemáticos essenciais (soma, subtração, multiplicação, divisão).
  * **Relacionais:** Comparações cruciais (`>`, `<`, `==`, `!=`).
  * **Lógicos:** Conectivos para regras complexas (`AND`, `OR`, `NOT`).

### 4. 🐍 Python (Em Evolução 🔥)
Agora o jogo ficou sério! Estou aplicando toda a minha base lógica na linguagem que move a inteligência artificial e a automação de dados. Até o momento, já domino:
* Entrada e saída de dados de forma dinâmica (`input` e `print`).
* Tratamento de tipos de dados (`float`, `int`, `string`).
* **Estruturas de Repetição Contínuas:** Loops com a estrutura `while` (enquanto).
* Criação de lógicas de segurança (como travar divisões por zero ou criar condições de parada em sistemas).
